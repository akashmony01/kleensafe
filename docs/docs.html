Change Logo

Its very easy to change your logo.
All you have to do is open your desired html file on a code editor and change the source/name of the logo to your logo.

Inside any file.html search for header element and follow this next code 
                                            

<a class="logo-link" href="#"><img class="logo-img light-logo" src="assets/Images/logo/logo-light.png" alt="logo">
    <img class="logo-img  dark-logo" src="assets/Images/logo/logo-dark.png" alt="logo">
</a>

Backgrounds

Image Background

To change the background image for any page you have to change the attrbiute data-bg-img value in the .overlay-photo-image-bg element to the image link that you want to place as a background

NOTE: the .overlay-photo-image-bg is the element that hold the bg image of the section

css/style.css 
                                            
/****************** INDEX 6 ***********************/

<div class="overlay-photo-image-bg" data-bg-img="assets/Images/hero/hero-1.jpg"></div>

Slider Background

To change the Background Images on any Index.hmtl file that has a hero section slider all you have to do is change the URL link to the image that you want to insert as a background
as the following example.

Any index.html that have a hero section slider 
                                            
<!-- Start of Slide 1 -->
<div class="swiper-slide " >

        <div class="overlay-color"></div>
    </div>

    <!-- Start of Slider text and cta Content -->
    <div class="container"> 
        <div class="hero-text-area   ">
            ...
        </div>
    </div>
        <!-- End of Slider Content -->
 </div>

<!-- End of Slide 1 -->

Change Header

To change the header, you have to replace the current header with the header that you want. For example:

file-with-current-header.html 
                                            
<!-- =============== Start of Header-basic Navigation =============== -->
<header class="header-basic">
....
....
</header>
<!-- =============== End of Header-1 Navigation =============== -->
file-with-current-header.html 
                                            
<!-- =============== Start of Header 1 Navigation =============== -->
<header class="header-1">
....
....
</header>
<!-- =============== End of Header 1 Navigation =============== -->

Lets say you want to replace the current header with the header-2. Open any file on the demos and grab the html structure of the header and put it on the page you want that header on
Like the examples below

header2.html 
                                            
<!-- =============== Start of Header-2 Navigation =============== -->
<header class="header-2">
....
....
</header>
<!-- =============== End of Header-2 Navigation =============== -->

Copy everything from the 'Start of Header 2' till the 'End of Header 2' with everything that is in it and replace with the whole code of your current header (as shown on the previous example).

Contact Form

Contact form will use php to send emails. So open send-mail.php file that can be found on the php folder. Change the current email "youremail@example.com" to your email address and you are ready to go.


NOTE: Contact form require web server to send emails. Emails can go to your spam/junk folder.

php/email.php 
                                            
########################################################################
########################################################################

$to             = "youremail@example.com"; // Change with your email address
$sub            = $subject; //   define email subject
$body           = <<<EOD
                    <strong>Name:</strong> $name <br>
                    <strong>Email:</strong> <a href="mailto:$email?subject=feedback" "email me">$email</a> <br> <br>
                    <strong>Message:</strong> $message <br>
                    EOD;
                    //Must end on first column
	
$headers        = "From: $name <$email>\r\n";
$headers        .= 'MIME-Version: 1.0' . "\r\n";
$headers        .= 'Content-type: text/html; charset=UTF-8 ' . "\r\n";

Google Map

1- to change the google map to your location simply open YOUR google map location on any browser ang copy the embede code of the loaction you need

2- paste the empeded code link on the src attribute of the iframe element as on the below example. And thats all.

js/custom.js 
                                            
/*----------------------------------------------------

----------------------------------------------------*/
<div class="map-box pt-5 mt-5">
        <div class="mapouter">
